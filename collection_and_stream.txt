🚀 Java Stream API – Complete Answers
🔹 1. What is Stream API?
Stream API (introduced in Java 8) is used to process collections of data in a functional and declarative way.

👉 It allows operations like:

filtering

mapping

sorting

reducing

list.stream().filter(x -> x > 10).toList();
🔹 2. Collection vs Stream
Feature	Collection	Stream
Storage	Stores data	Doesn’t store data
Nature	Eager	Lazy
Reuse	Yes	No
Traversal	Multiple times	One-time
🔹 3. What is Lazy Evaluation?
Operations are executed only when a terminal operation is called.

list.stream()
    .filter(x -> x > 10)   // not executed yet
    .count();              // executed here
🔹 4. Intermediate vs Terminal Operations
Intermediate (returns Stream)
filter()

map()

sorted()

Terminal (returns result)
forEach()

collect()

count()

🔹 5. forEach() vs map()
forEach()	map()
Terminal	Intermediate
No return	Returns new stream
Used for printing	Used for transformation
list.stream().map(x -> x * 2).toList();
🔹 6. filter()
Used to select elements based on condition

list.stream()
    .filter(x -> x % 2 == 0)
    .toList();
🔹 7. reduce()
Used to combine elements into single result

int sum = list.stream().reduce(0, (a, b) -> a + b);
🔹 8. map() vs flatMap()
map()
Transforms each element

flatMap()
Flattens nested structure

List<List<Integer>> list = ...
list.stream()
    .flatMap(x -> x.stream())
    .toList();
🔹 9. distinct()
Removes duplicates

list.stream().distinct().toList();
🔹 10. sorted()
list.stream().sorted().toList();
Custom:

list.stream().sorted((a,b) -> b-a).toList();
🔹 11. limit() and skip()
list.stream().limit(5).toList(); // first 5
list.stream().skip(5).toList();  // skip first 5
🔹 12. collect()
Converts stream into collection

list.stream().collect(Collectors.toList());
🔹 13. Collectors class
Utility class to perform mutable reduction

Examples:

Collectors.toList()
Collectors.toSet()
Collectors.groupingBy()
Collectors.joining()
🔹 14. findFirst() vs findAny()
findFirst()	findAny()
Returns first element	Returns any element
Ordered streams	Better for parallel
🔹 15. Optional class
Used to avoid NullPointerException

Optional<Integer> val = list.stream().findFirst();
val.ifPresent(System.out::println);
🔹 16. peek()
Used for debugging

list.stream()
    .peek(x -> System.out.println(x))
    .toList();
🔹 17. Parallel vs Sequential Stream
list.parallelStream()
Sequential	Parallel
Single thread	Multiple threads
Slower	Faster (large data)
🔹 18. ForkJoinPool
Parallel streams use ForkJoinPool.commonPool()

👉 It splits tasks into smaller subtasks

🔹 19. When to use Parallel Streams?
Use when:

Large data

Independent operations

CPU intensive

Avoid when:

Small data

Shared mutable state

🔹 20. Side Effects in Streams
Avoid modifying external variables

❌ Bad:

list.stream().forEach(x -> sum += x);
🔹 21. Stateful vs Stateless Lambda
Stateless	Stateful
No dependency	Depends on external state
Safe	Unsafe
🔹 22. Thread Safety
Streams are:

Not thread-safe by default

Parallel streams need stateless operations

🔹 23. Spliterator
Used to split data for parallel processing

🔹 24. Short-circuiting operations
Stop processing early

Examples:

findFirst()

anyMatch()

🔹 25. Can we reuse a Stream?
❌ No

Stream s = list.stream();
s.count();
s.count(); // Exception
🔹 26. map() vs collect()
map()	collect()
Transform data	Convert to collection
Intermediate	Terminal
🔥 Bonus Coding Examples
✔ Grouping
list.stream().collect(Collectors.groupingBy(x -> x));
✔ Frequency
list.stream()
    .collect(Collectors.groupingBy(x -> x, Collectors.counting()));
✔ Join Strings
list.stream().collect(Collectors.joining(","));

✅ 1. Find Even Numbers
List<Integer> list = List.of(1,2,3,4,5,6);

        .filter(x -> x % 2 == 0)
        .toList();

for(int x : list){
    if(x % 2 == 0) result.add(x);
}

        .toList();

Set<Integer> set = new HashSet<>(list);

✅ 6. Frequency of Each Element
List<String> list = List.of("a","b","a","c","b","a");

Map<String, Long> freq = list.stream()
    .collect(Collectors.groupingBy(x -> x, Collectors.counting()));
✅ 7. Convert List to Map
Map<Integer, Student> map = students.stream()
    .collect(Collectors.toMap(s -> s.id, s -> s));
✅ 8. Group Strings by Length
List<String> list = List.of("java","python","c","js");

Map<Integer, List<String>> result =
    list.stream().collect(Collectors.groupingBy(String::length));
✅ 9. First Non-Repeated Character
String str = "aabbcde";

Character ch = str.chars()
    .mapToObj(c -> (char)c)
    .collect(Collectors.groupingBy(c -> c, LinkedHashMap::new, Collectors.counting()))
    .entrySet()
    .stream()
    .filter(e -> e.getValue() == 1)
    .map(e -> e.getKey())
    .findFirst()
    .get();
✅ 10. Find Duplicate Elements
List<Integer> list = List.of(1,2,3,2,4,1);

Set<Integer> seen = new HashSet<>();

Set<Integer> duplicates = list.stream()
    .filter(x -> !seen.add(x))
    .collect(Collectors.toSet());

    .getKey();
